# Installation Guide

## Steps to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/apiz23/bus-ticketing-system.git
   ```

2. Open your terminal, CMD, or Powershell in the cloned folder.

3. Run the script:
   - On CMD:
     ```cmd
     run.bat
     ```
   - On Powershell:
     ```bash
     ./run.bat

   - On Mac:
     ```bash
     chmod +x run.sh
     ./run.bat
